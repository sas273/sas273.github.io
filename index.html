Site Structure

  • index.html — this home page.
  • lessons/grade3-norse.html — sample Grade 3 lesson page.
  • lessons/lesson-template.html — copy for new lessons.
  • assets/css/style.css — minimal styles for index page.
  • .nojekyll — disables Jekyll processing (so folders work as-is).

Getting Started (GitHub Pages)

  1. Create a free GitHub account (if you don’t have one).
  2. Create a new repository named username.github.io (replace username with your GitHub username).
  3. Upload the files from this starter (drag & drop the whole folder).
  4. Commit the changes. Your site should appear at https://username.github.io within a minute.

Prefer a project site (not username.github.io)? Enable Pages in repository settings, set branch to main and root folder.

Next Steps

  • Duplicate lessons/lesson-template.html for each new lesson.
  • Update links and navigation as your catalog grows.
  • Later, connect a custom domain in GitHub Pages settings.
